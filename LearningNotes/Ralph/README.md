## 🚀 推荐学习路径

### 路径 1：快速上手（适合赶时间的用户）
1. 阅读 [快速参考](Ralph快速参考.md) 的"安装命令"和"项目初始化"部分（5 分钟）
2. 跟随 [实践指南](Ralph实践指南.md) 的"第一次使用 Ralph"（20 分钟）
3. 开始使用 Ralph，遇到问题查阅 [快速参考](Ralph快速参考.md)

**总时长**：约 30 分钟可以开始使用

### 路径 2：深度学习（适合想要精通的用户）
1. 完整阅读 [项目理解文档](Ralph项目理解文档.md)（40 分钟）
2. 跟随 [实践指南](Ralph实践指南.md) 完成至少一个实战案例（2-4 小时）
3. 将 [快速参考](Ralph快速参考.md) 保存为浏览器书签，随时查阅

**总时长**：第一天约 3-5 小时，之后持续实践

### 路径 3：团队培训（适合团队推广）
1. 技术负责人先完成"路径 2"（深度学习）
2. 准备团队培训会议：
   - 讲解 [项目理解文档](Ralph项目理解文档.md) 的核心概念（30 分钟）
   - 演示 [实践指南](Ralph实践指南.md) 的一个实战案例（30 分钟）
   - 分发 [快速参考](Ralph快速参考.md) 作为团队手册
3. 在实际项目中试点使用，收集反馈
4. 迭代优化团队的 Ralph 使用流程

**总时长**：团队培训 1 小时 + 试点 1-2 周

## 🔧 快速开始（5 分钟）

如果你只想最快速地开始使用 Ralph：

```bash
# 1. 安装 Ralph（一次性）
git clone https://github.com/frankbria/ralph-claude-code.git
cd ralph-claude-code
./install.sh

# 2. 在你的项目中启用
cd your-project
ralph-enable

# 3. 开始自主开发
ralph --monitor
```

详细步骤请查看 [快速参考](Ralph快速参考.md#安装命令)。
